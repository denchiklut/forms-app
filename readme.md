# Установка проекта
1. Склонируйте проект ик  
`git clone https://your_username@bitbucket.org/denchik_front/forms-app.git`
2. Установите необходимы зависимости  
`npm install`
3. Для разработки используйте команду:  
`npm run watch-poll`
4. Для деплоя на сервер команду  
`npm run production`
